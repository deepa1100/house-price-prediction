# 🏠 House Price Prediction Web App

## 📌 About Project
This is a Machine Learning based web application that predicts house prices using user input features such as total square feet area, location, number of bathrooms, and BHK. It also includes a login system, prediction results, and data visualization graphs.

## 🧠 Technologies Used
- Python
- Flask
- HTML, CSS
- Machine Learning (Regression Model)
- Pandas, NumPy, Scikit-learn
- Matplotlib

## 📊 Features
- User login system
- Home page with input fields:
  - Total Square Feet (sq ft)
  - Location
  - BHK (Bedrooms, Hall, Kitchen)
  - Number of Bathrooms
- House price prediction using ML model
- House Price vs Total Square Feet graph
- House Price Distribution graph
- Download prediction report as PDF
- Logout functionality

## 🎯 Objective
To develop a machine learning-based web application that predicts house prices accurately and provides visual insights through graphs and downloadable reports.
